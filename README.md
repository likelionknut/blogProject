# 충주팀 멋쟁이 사자처럼 스터디 logProject


### 참여한 팀원 명단
	- 대표 : 명평윤
	- 부대표 : 조경록
	- 교육 담당 : 박상돈, 임현석


### 작품 개요
	- 멋사 7기 운영진 교육 final team project
	- 사이트 회원가입 및 로그인 기능
	- 블로그 게시물 게시 및 조회 기능
	- 포트폴리오 게시 및 조회 기능
	- 부트스트랩을 이용한 디자인 수정
	- OpenWeatherMap API를 사용해 날씨 정보 제공


### 기술스택
	- Django
	- Python
	- Bootstrap Theme (https://mdbootstrap.com/)
	- Plain HTML/CSS/js
